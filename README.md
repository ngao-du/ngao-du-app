# Ngao Du

#### Ngao du đó đây và thưởng thức ẩm thực

### Technology

- Flutter: 3.22

### Screen

1. Home page: Danh sách địa điểm, có tính năng search, filter và order.
2. Detail page: Chi tiết về 1 địa điểm:
    - Tên quán, địa chỉ.
    - Thời gian mở cửa.
    - Maps.
    - Số sao, bình luận.
    - Món nổi bật.
    - Kỷ niệm với quán.
3. Đăng ký/Đăng nhập
4. Quán sẽ đi.
